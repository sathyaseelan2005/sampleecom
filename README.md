# Lumina E-commerce Sample

A premium, futuristic e-commerce landing page built with Vanilla HTML, CSS, and JavaScript.

## Features
- **Dark Mode / Cyberpunk Aesthetic**: Modern design with neon accents and glassmorphism.
- **Responsive Design**: Works on desktop and mobile.
- **Dynamic Product Grid**: Products rendered via JavaScript.
- **Functional Cart**: Add/remove items, calculate total.
- **Animations**: Smooth scrolling, hover effects, entrance animations.

## How to Run
Simply open **`index.html`** in your web browser.

## Technologies
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (ES6+)
- Ionicons (Icons)
- Google Fonts (Outfit)
